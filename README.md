## 📌 Sobre o projeto

O **Sistema de Ranking de Jogadores** é uma aplicação web desenvolvida para gerenciar campeonatos e acompanhar o desempenho dos participantes de forma simples, moderna e intuitiva.

A aplicação permite cadastrar jogadores, registrar resultados de partidas, calcular automaticamente a pontuação e organizar uma classificação dinâmica baseada no desempenho de cada participante.

O projeto possui uma interface moderna no estilo **dashboard**, com design responsivo, efeitos de transparência (*glassmorphism*) e elementos visuais para melhorar a experiência do usuário.

---

## 🚀 Funcionalidades

✅ Cadastro de jogadores  
✅ Registro de vitórias, empates e derrotas  
✅ Cálculo automático de pontuação  
✅ Ranking atualizado automaticamente  
✅ Ordenação por melhor desempenho  
✅ Destaque dos primeiros colocados 🥇🥈🥉  
✅ Exclusão de jogadores  
✅ Reinicialização do campeonato  
✅ Pesquisa de jogadores  
✅ Salvamento automático no navegador  
✅ Painel de estatísticas  
✅ Gráfico de desempenho dos jogadores  

---

## 🎯 Objetivo

O objetivo deste projeto é criar uma ferramenta simples para organização de campeonatos, permitindo acompanhar resultados e desempenho dos jogadores sem a necessidade de sistemas complexos.

Além disso, o projeto demonstra conceitos importantes do desenvolvimento web:

- Manipulação do DOM
- Criação de interfaces responsivas
- Programação orientada a eventos
- Armazenamento local no navegador
- Organização de componentes visuais
- Desenvolvimento de aplicações interativas
